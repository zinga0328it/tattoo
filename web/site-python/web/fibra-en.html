Architettura del Sistema

Il sistema utilizza un'architettura distribuita innovativa:

  • Frontend Pubblico: Interfaccia web responsive ospitata su Apache con HTTPS
  • Proxy Sicuro: Apache ProxyPass che inoltra richieste alla rete Yggdrasil
  • Backend Isolato: FastAPI che ascolta solo su interfaccia Yggdrasil privata
  • Database Sicuro: PostgreSQL accessibile solo dal backend isolato
  • Bot Telegram: Servizio separato per notifiche e comandi mobili
